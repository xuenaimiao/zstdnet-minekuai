# ZstdNet 使用文档（新手友好）

> 这是一份**写给完全没有基础的人**看的说明书。不需要懂网络、不需要懂代码，照着一步步做就行。

ZstdNet 是一个 Minecraft Java 版的「**省流量**」工具：它把你和服务器之间来回传的数据**压缩**一下再发，
能明显降低公网带宽占用，尤其适合机械动力服、大型整合包服、走内网穿透 / FRP 的联机。

## 30 秒速览

- **要压缩，玩家和服务器最好都装**（只装一边就没法压）。
- 服主默认配置下，**玩家还是连原来那个地址**，不用记新端口。
- 一切都是**默认安全**的：装上但不动配置，行为和没装时一模一样；没装的玩家照常进服，互不影响。
- 想看有没有生效：进游戏打 `/zstdhud` 调出面板，看 **Ratio**（越低越省）。
